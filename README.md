## Backend

- Предварительно запустить сервер MongoDB
- Прописать подключение в .env при отличном от стандартного

1. cd backend
2. npm i
3. npm run build
4. npm run start:prod
- http://localhost:3000

## Frontend:
1. cd frontend
2. npm i
3. npm run build
4. npm run preview
- http://localhost:4173
